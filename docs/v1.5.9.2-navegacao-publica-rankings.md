# v1.5.9.2 — Navegação pública de Rankings

## Objetivo

Adicionar a página pública de Rankings na navegação global do site, posicionada depois de Comunidades e antes de Transferências.

## Alterações

- Adicionado link `Rankings` nos menus principais das páginas HTML que já tinham navegação pública.
- O link aponta diretamente para `/rankings/rankings.html` usando caminho relativo correto conforme a pasta de cada página.
- A ordem aplicada no menu principal ficou:
  - Comunidades
  - Rankings
  - Transferências
- Atualizado o reconhecimento de pasta pública nos helpers de autenticação para incluir `/rankings/`.

## SQL

Nenhum SQL novo é necessário nesta etapa.

## Teste recomendado

1. Abrir páginas públicas principais como Home, Torneios, Comunidades, Perfis, Equipes e Transferências.
2. Confirmar que o menu mostra `Rankings` entre `Comunidades` e `Transferências`.
3. Clicar em `Rankings` e confirmar abertura direta de `/rankings/rankings.html`.
4. Confirmar que a página de Rankings continua carregando jogadores/equipes conforme a v1.5.9.1 patch1.

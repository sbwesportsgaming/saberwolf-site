# v1.6.72.5 — Resumo público Team Battle League 4v4

Este patch adiciona uma camada de resumo público para o formato planejado **Team Battle League 4v4**.

## O que entrou

- helpers para montar resumo público de confronto entre equipes;
- resumo público das partidas principais e da partida extra;
- resumo de rodada;
- resumo de divisão;
- resumo geral da liga;
- totais públicos de divisões, equipes, rodadas e confrontos;
- preparação para futuras páginas públicas do formato 4v4.

## Regra mantida

O formato continua como **em preparação** e segue bloqueado para criação real.

## Arquivos alterados

- `js/tournaments/team-battle-league.js`
- `docs/releases/v1_6_72_5_team_battle_public_overview.md`

## Observação

Este patch não altera Supabase, Auth, RLS, criação real de torneio, inscrições, check-in, bracket, Ranking Global ou Admin Master.

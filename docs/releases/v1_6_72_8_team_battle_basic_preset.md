# v1.6.72.8 — Preset básico Team Battle League 4v4

Este patch adiciona uma camada técnica para separar oficialmente os dois modos do formato:

- Team Battle League 4v4 básica: uma divisão única.
- Team Battle League 4v4 avançada: várias divisões.

## Ajustes

- Criado preset técnico da liga básica com `division-unica`.
- Criado helper para gerar liga básica a partir de uma lista de equipes.
- Criado helper para gerar liga avançada com múltiplas divisões.
- Criado helper de seleção por modo.
- Criado relatório de prontidão específico para a liga básica.
- Ajustado catálogo do formato para reforçar que a primeira versão funcional deve priorizar a divisão única.

## Observação

O Team Battle League 4v4 continua em preparação e ainda não está liberado para criação real.

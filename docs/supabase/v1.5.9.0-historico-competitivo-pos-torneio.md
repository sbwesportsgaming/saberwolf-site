# v1.5.9.0 — Histórico competitivo pós-torneio

## Objetivo

Primeira camada para fazer torneios reais finalizados alimentarem os perfis dos jogadores.

Esta etapa ainda não cria ranking global oficial, temporadas oficiais ou pontuação definitiva da SBW. Ela apenas lê dados já existentes no Supabase:

- `tournaments.status = completed`
- `tournaments.settings.finalResults`
- `tournament_participants`

Com isso, o perfil público e o painel do usuário passam a exibir histórico/medalhas derivados dos resultados finais.

## O que foi adicionado

- Histórico competitivo gerado a partir de torneios finalizados.
- Medalhas automáticas para Top 8:
  - 1º: ouro/campeão, 100 pts
  - 2º: prata/vice, 75 pts
  - 3º: bronze, 50 pts
  - 4º: top 8, 30 pts
  - 5º: top 8, 20 pts
  - 6º: top 8, 15 pts
  - 7º: top 8, 10 pts
  - 8º: top 8, 5 pts
- Perfil público agora combina medalhas locais com medalhas vindas de torneios finalizados.
- Histórico público agora combina registros locais com resultados finais reais.
- Painel `/perfis/meu-perfil.html` também exibe essas medalhas/históricos derivados.
- `playerToMatchPlayer()` passou a carregar `participantId`, `profileId`, `playerSlug` e `authUserId` nas próximas estruturas geradas, melhorando o vínculo futuro com perfis.

## SQL

Não precisa rodar SQL novo nesta etapa.

A etapa usa as tabelas e policies já criadas nas fases anteriores:

- `public.tournaments`
- `public.tournament_participants`

## Como testar

1. Deixe o Supabase ligado em `js/supabase/supabase-config.js`.
2. Use um torneio real já finalizado na v1.5.8.9.
3. Abra o perfil público do jogador participante:

```txt
/perfis/perfil.html?id=slug-ou-id-do-perfil
```

4. Confirme se aparecem:

- medalha/conquista do torneio finalizado;
- histórico em Títulos, Pódios ou Torneios;
- estatísticas do hero atualizadas quando o perfil não tiver números maiores já cadastrados.

5. Abra também:

```txt
/perfis/meu-perfil.html
```

6. Confirme se as abas `Medalhas` e `Histórico` também mostram os dados pós-torneio.

## Observação

Esta etapa é uma ponte entre torneios finalizados e perfis. Ranking oficial SBW, temporada do organizador, ranking de equipe e auditoria de pontuação entram em etapas próprias depois.

# v1.5.8.8 — Resultados reais no detalhe público

Etapa focada em mostrar publicamente os resultados lançados pelo organizador na v1.5.8.7.

## Ajustes principais

- O detalhe público reconhece o status `in-progress` como **Em andamento**.
- A seção de estrutura mostra também:
  - quantidade de resultados lançados;
  - total de partidas jogáveis;
  - data/hora do último resultado salvo.
- Partidas públicas exibem:
  - placar salvo;
  - vencedor destacado;
  - status visual da partida, como `Pendente`, `Finalizada`, `Em andamento` ou `BYE`.
- Histórico de partidas de liga/grupos passa a destacar vencedores e status.
- Tela de criação/gerenciamento do organizador recebeu navegação no canto superior direito para voltar à aba pública de **Torneios**.
- Perfil público do organizador também recebeu atalho superior para **Torneios**.

## SQL

Não há SQL novo nesta etapa.

## Teste recomendado

1. Abrir um torneio real já com estrutura gerada.
2. Lançar pelo menos um resultado no painel do organizador.
3. Abrir:

```txt
/torneios/detalhe-torneio.html?id=slug-do-torneio
```

4. Confirmar se aparecem:

- status do torneio como **Em andamento**;
- placar da partida;
- vencedor destacado;
- card da estrutura mostrando resultados lançados e último resultado.

5. Abrir:

```txt
/torneios/create-tournament/criar-torneio.html
```

6. Confirmar se o botão/atalho **Torneios** aparece no canto superior direito do painel.
